Подключайте ноутбук с помощью одного кабеля USB-C

Этот дисплей Philips оснащен разъемом USB Type-C для подачи питания. С интеллектуальным и гибким управлением питанием вы сможете напрямую заряжать совместимое устройство. Тонкий двусторонний интерфейс USB-C обеспечивает подключение посредством одного кабеля. Вы можете просматривать видео высокого разрешения, одновременно с этим заряжая совместимое устройство.
